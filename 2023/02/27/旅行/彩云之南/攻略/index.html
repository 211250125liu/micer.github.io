云南旅行攻略

发表于2023-02-27|更新于2023-03-01|旅行
|阅读量:

具体行程：
2.1 柳州出发，昆明南转车，五点半到达大理，公交车到达客栈，八点半躺在床上，十点多吃饱（石板烧），逛了一下大理古城。
2.2 租车环洱海，八点半起床，九点出发，（其实建议早起一点在海西这边看日出的），九点半左右到达喜洲古镇，逛逛挺有意思的，然后继续顺时针转，沿途可以靠边停车看洱海，双廊古镇、风情岛人太多，没停，小普陀停了，停车费10r，就一个小破岛，看看海鸥挺好，岛上没去，猜也不好玩。罗荃半岛没进去，人多。圣托里尼进了，就是一个房地产项目，走走看看过瘾就行了，然后继续转，绕过市区回大理古城，开过了兴盛大桥，咋说，就这样吧，没太大感觉（手动狗头），晚上吃的古城手抓饭感觉一般，特色肯定有，广西小绵羊感觉吃不习惯。逛了大理人民路，很热闹，很多小姐姐披那种（说不出来是啥），很好看，然后沿途很多酒吧，还有那种占卜的，不是很信，没看。
2.3 早上八点多起床，打车去崇圣寺三塔（离古城很近，10r打车），进去了就是三个塔，很高，不能进去，然后后面就是崇圣寺，很大，要走的很多，平底鞋软底很重要，在崇圣寺后面的高楼上可以看见苍山洱海，这是大理唯一的五星级景区哇，学生票半价（37.5，现场购买）。反正看看挺有意思的这地方。逛完去了KFC，KFC完打车前往龙龛码头，当时封闭了，远看洱海吧，然后划重点了，一定租车（自行车），可以边骑边看海，想停就停下来一下。走路属实痛苦，然后其实很多地方都有S湾，因为走不动了就没去磻溪村。直接回了客栈躺。晚上吃了灰太狼石板烧，吃完没有选择一杯大理（因为网上评价很烂），选了一个酸奶西米的店，也有那个大理的杯套，拍照很出片，也挺好喝的。然后逛了一下洋人街，我愿称之为酒吧街，然后为了满足罗√的烧鸡愿望，去买了烧鸡，还买了释迦，云南水果是真的便宜哇，好吃。晚上大家一起拉肚子，烧鸡事件。。。
2.4 起床去吃了一碗米线，评价为辣鸡，然后坐车去丽江。大概车开出30min这样可以看到洱海，看到的时间在一分钟之内。下午到的丽江，打车去丽江古城（住在那边），住的木茗别院（不推荐，花洒有问题，屋外环境还可以，屋内空调开不了，很冷，反正差评。大理住的十里一栈，还可以，可以再去住）晚上丽江古城，去吃了腊排骨/土鸡火锅，一般般吧就，然后逛了逛丽江古城，基础设施比大理好，很多公共卫生间，然后买了鲜花饼，记得买热的，现烤的，贵点但是好吃。去了忠义市场，夜市街，感觉跟别的地方差不多，丽江古城也就是逛逛逛，没有特别推荐的，看个人感觉吧。睡，晚上洗澡花洒差评，冷死我了。然后就是说，云南昼夜温差很大，昆明>大理>丽江
2.5 玉龙雪山，雪山101专线，15r，送你到景区，然后进山费50r（学生票半价），然后其他的就是各种索道票，索道票没有半价，理解为交通费。我抢到大索道票了，骄傲一下，退了跟大伙走，怕死。云杉坪开发程度高一点，氧气这个，吸氧气挺爽的，嘿嘿，云杉坪可以看到雪山，还有牛羊，很多人在这拍婚纱照。云杉坪下来到蓝月谷，感觉也没那么蓝，水不能喝拍照出片这种，，，不会拍，但是有一说一，很好看是真的。然后下一步，牦牛坪，很原始，索道很高，别慌死不了，上去之后栈道有点小问题哇，慢点不要踩空，软底鞋再一次重要。有个雪花寺，有个喇嘛，反正不信教，尊重就好，敬畏一点，毕竟海拔4k，有那种经幡，会拍照肯定很nice，这里的海拔就有点高了，不要剧烈运动，多吸吸氧，不能带上动车，不吸就浪费了。然后坐车会丽江市区。晚上吃了个丽江特色菜，还可以，但是一盘菜一盘菜上，来一盘就没一盘。继续逛丽江古城，然后就玩玩玩。晚上去了酒吧，建议美团看好哇，里面的小贵，然后选个唱歌好听的酒吧很重要，悠着点少喝点，清吧不建议喝醉。
2.6 早上坐车去丽江站，公交车，丽江站看起来挺好看的，中午到达昆明。酒店评价就是很实惠，免费早餐点个赞。下午去了昆明老街，那个纸片楼咋说，感觉就那样，xswl，接着找个店吃了过桥米线，仪式感很强，吃起来还行就是说。然后晚上逛的是一些昆明的夜市，网上搜搜可以找到的。就这样，没啥很特别的。
2.7 今天起来去了海埂大坝，带面包去哈，去喂海鸥，很多，很好看，但是也会天降正义不是（手动狗头），就拿着面包伸手高一点，会有海鸥来吃，你抓稳点就能被多咬几口，不然就是整个被叼走。海埂大坝过去有个西山，没去，不懂咋样，去了海洪湿地公园。沿途风很大，吹着很爽，浪也很大，注意点别进滇池了。在海洪湿地公园划船了，就那种树木之间有水，比较浅，两人一组，90r/h，划船方法，一人一个船桨，两个人一起同频就是往前，想往右就左边的船桨发力，左边一样，注意点别搞裤子太湿。然后去了斗南花市，真的很多花，以及和花有关的，不算贵，但是一定要好好跳，选那种开了的好点，个人感觉吧。晚上吃的烤肉，还行，小贵，晚上买了点嘉华鲜花饼，就这样吧，不如现烤现吃的。
2.8 出发贵阳，中午到的，去吃了丝娃娃，就那样没感觉，酸汤很酸吃不习惯，然后吃的很辣也不习惯，可伶我广西小绵羊。然后特色是那个肠旺面，不吃内脏和血，拒绝了。吃饱去了黔灵山，猴子很多，会跟你抢东西那种，所以不要拿塑料袋，以及带好防身物品。里面有个动物园，有熊猫还有其他一些动物。出来后吃的是，一种火锅，底料是大的那种红豆吧，和软哨（据说是肥瘦肉一起榨油完剩下来的肉），汤有点辣，感觉一般。晚上去了甲秀楼，进不去，外面看着也一般，just so so
2.9 睡到自然醒，然后去了贵州博物馆，一般，反正看过南京博物院之后，对比起来就不咋的。没啥评价的，中午吃的KFC，不辣，真好。下午去地质博物馆，里面有那种贵州一些地形景点的缩略图，这倒挺有意思。评价是还不错，晚上吃某种汤，意想不到的是还有辣椒，说汤里面那些东西要蘸辣椒吃，绷不住了，我再也不来了，这吃的东西要命啊
2.10 八点半的飞机飞南京，南航的空中伙食挺好吃的，但是空姐都戴口罩看不出好不好看，但是服务态度挺好，机场回学校两小时。南京地铁修的狗啊，中间换乘走了差不多1km。十二点半回到学校，终于时隔两个月回来了。
